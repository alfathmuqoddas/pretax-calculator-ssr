# Pre-Tax Income Calculator (SSR)

This is a simple calculator that calculates how much the pre tax income amount needed to generate n amount of after-tax income. There's so many income tax calculators out there already, but not many of them provide a simple and deterministic way to calculate pre-tax income. Hope this app helps your financial planning. This is a SSR version of the app, using Astrojs and Cloudflare Adapter.
